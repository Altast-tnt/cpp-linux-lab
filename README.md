# 🐧 C++ & Systems Lab

Добро пожаловать в мою лабораторию C++.
Этот репозиторий — живой журнал моего погружения в системное программирование, разработки под Linux и экспериментов с C++ в различных сферах (от ядра системы до GameDev).

### 🎯 Цели репозитория
1.  **Deep Dive:** Углубленное изучение C++ (Standard Library, Memory Management).
2.  **Linux Internals:** Работа с системными вызовами, архитектурой ОС и ядром.
3.  **Fun & Games:** Разработка небольших игр и графических приложений для практики.

### 🗂 Навигация
*   `cources` - раздел с изучением C++ и связанных технологий
*   `QML` - изучение QML
*   `dubaitTest` - небольшое тестовое задание для "Дубайт"

### 🛠 Инструментарий
*   **Languages:** C++, C, Bash, QML
*   **OS:** Fedora Linux / Linux General
*   **Build Systems:** CMake, Makefiles, GCC/Clang
*   **Tools:** GDB, Valgrind, Vim/NeoVim
